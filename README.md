# 🏫 Student Hostel Management System
Group Assignment for IMS511 - Problem Solving and Program Design.

## 📝 Overview
A console-based student hostel management system implemented in C++ to manage student registrations and room allocations effectively.

## ✨ Features
- Add new students with contact and demographic details
- Display all registered students
- Display hostel statistics (Total Students & Average Age)
- Search student records
